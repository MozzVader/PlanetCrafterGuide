# Guia de Planet Crafter

Guia completa en espanol para **Planet Crafter** - el juego de supervivencia y terraformacion espacial de Miju Games.

## Contenido

- Introduccion al juego
- Primeros pasos (guia de inicio)
- Indices de terraformacion (O2, Calor, Presion, Biomasa)
- Las 9 etapas de terraformacion
- Recursos y minerales
- Maquinas clave organizadas por categoria
- Consejos avanzados (Pro Tips)
- Cofres dorados y secretos

## Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- JavaScript vanilla (Intersection Observer, Canvas API)

## Deploy

Esta guia esta disenada para ser alojada en **GitHub Pages** como un sitio estatico.
